# Kasane

A simple open-source manga reader app created using Kodular because why not and it fun

## Features include

<div align="left">

- [MangaDex](https://mangadex.org/) manga reader for Android 7+
- History tracking


<div align="center">

## Limitation (Will be added in future)

<div align="left">
  
- No Favourate feature
- No offline reading
